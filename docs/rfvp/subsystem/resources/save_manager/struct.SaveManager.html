pub fn pending_save_capture(&self) -> Option<(u32, u32, u32)>

If either a SaveCreate(3, nil/int) requested preparation of an in-memory payload (original engine: local_saved), or a SaveWrite requested a fallback capture, return (slot, thumb_w, thumb_h) so the renderer can read back the current frame.

Special slot value u32::MAX indicates “prepare local_saved” (no file write).

Source

pub fn request_prepare_local_savedata(&mut self)

Request preparing an in-memory save payload (original engine: local_saved).

This should be triggered when the save/load UI is opened (scripts call SaveCreate(3, nil)), so subsequent SaveWrite(slot) can commit without capturing the menu overlay.
