pub fn force_reveal_all_non_suspended(&mut self)

Force-reveal all characters for every loaded, non-suspended slot.

This is used to implement the original engine’s behavior where holding Ctrl (or issuing ControlPulse) makes the current line render immediately.

Note: the original engine skips text updates entirely while a TextPause is active; we preserve that by ignoring suspended slots here.
